## 🎯 عن المشروع (Overview)
مستوحى من الأناقة والاحترافية، يقدم **Tiger-X Invoice Generator** واجهة مستخدم (UI) فاخرة ومظلمة (Dark Mode) مدمجة باللون الذهبي الملكي لتناسب الشركات الناشئة، الاستوديوهات الرقمية، والفريلانسرز الذين يبحثون عن مظهر احترافي أمام عملائهم.

---

## 💎 المميزات الجبارة (Features)
* 🏢 **إدارة كاملة للبراند:** إمكانية رفع لوجو الشركة وتخزين بيانات الاتصال تلقائياً في المتصفح (`LocalStorage`).
* 💵 **تعدد العملات والضرائب:** دعم العملات العربية والعالمية مع حساب تلقائي لضريبة القيمة المضافة (VAT) والخصومات.
* ⏳ **تتبع حالة الفاتورة:** نظام أختام ذكي يتغير ديناميكياً حسب حالة الفاتورة (قيد الانتظار ⏳ - مدفوعة ✅ - ملغية ❌).
* 🖨️ **تصدير فوري وبدون أخطاء:** متوافق تماماً مع الطباعة والتصدير كملف PDF نظيف ومنسق.
* 📤 **مشاركة سريعة:** زر مدمج لنسخ ومشاركة رابط الفاتورة مع العميل مباشرة.

---

## 🛠️ كيف يعمل؟ (How it works)
1. افتح الأداة عبر المتصفح.
2. أدخل بيانات شركتك وبيانات العميل والخدمات.
3. اضغط على **"✦ إصدار الفاتورة ✦"** ليقوم النظام بتوليدها فوراً بالأسفل.
4. اضغط **طباعة** أو **مشاركة** لإرسالها للعميل.
